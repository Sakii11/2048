## 第9页：技术分析 — 自定义View绘制

### 为什么用Canvas而不是XML布局？

| 对比项 | XML布局 | Canvas绘制 |
|--------|---------|-----------|
| **动画控制** | 受限，只能操作整个View | 精确控制每个方块的位置、大小、透明度 |
| **性能** | 16个View嵌套，布局开销大 | 单次onDraw绘制所有元素，高效 |
| **灵活性** | 布局固定，动画有限 | 可实现任意形状、任意路径动画 |
| **内存** | 每个方块是一个View对象 | 只有数组数据，内存占用小 |

### BoardView绘制流程
```
onDraw(Canvas canvas)
  ├─ 绘制棋盘背景（圆角矩形）
  ├─ 绘制16个空格子（4×4网格）
  ├─ 遍历棋盘数组
  │   ├─ 滑动中：插值计算位置 → drawTileAt()
  │   └─ 静止/合并中：计算缩放 → drawTileAt()
  └─ drawTileAt()：圆角矩形 + 渐变 + 数字文字
```

### 背景动画模块 [StarsView.java]
- 30个几何粒子（圆形/圆角方块/菱形）
- 缓慢向上漂浮 + 水平摇摆 + 自转
- 6主题配色，颜色随主题切换

## 第10页：技术分析 — 性能优化

### 优化前的问题
- 每帧创建大量临时对象，GC频繁
- 动画回调竞态，旧动画干扰新动画
- 增强for循环每次创建迭代器

### 优化措施

#### 1. 对象缓存复用
- **Paint/Shader缓存**：12种方块渐变预创建，不复用GC
- **数字字符串缓存**：0~4096数字字符串预生成，避免String.valueOf
- **数组复用**：临时数组成员变量化，减少int[]/float[]分配
- **RectF复用**：tempRect成员变量，避免每帧new

#### 2. 算法优化
- **slideMap直接索引**：O(n)线性查找 → O(1)数组索引
- **RowResult复用**：每次合并不再创建新对象
- **原地反转**：reverse()创建新数组 → reverseInPlace()原地修改

#### 3. 代际思想解决动画竞态
```java
private int animGen = 0; // 代际计数器

public void animateMove(...) {
    final int gen = ++animGen; // 递增代际
    animator.addListener(new AnimatorListenerAdapter() {
        @Override
        public void onAnimationEnd(Animator a) {
            if (gen != animGen) return; // 代际检查，旧回调失效
            // ... 执行动画结束逻辑
        }
    });
}
```

### 优化效果
- 对象分配减少约 **40%~60%**
- GC频率显著降低
- 游戏运行更流畅

## 第14页：AI使用心得

### AI在项目中的应用

#### 1. 代码编写辅助
- **快速实现**：基础框架、模板代码快速生成
- **Bug修复**：定位问题原因，提供修复思路
- **代码优化**：性能瓶颈分析，优化方案建议

#### 2. 设计方案咨询
- **UI设计**：配色方案、布局结构建议
- **架构设计**：模块划分、技术选型讨论
- **动画设计**：动画效果方案，参数调优建议

#### 3. 知识获取
- **API查询**：快速了解Android API用法
- **原理讲解**：深入理解技术原理（如Canvas绘制、动画机制）
- **最佳实践**：学习行业标准的开发模式

### AI使用的体会

#### 优势
1. **效率提升**：减少重复劳动，聚焦核心逻辑
2. **知识扩展**：快速接触新的技术点和优化思路
3. **即时反馈**：有问题随时提问，快速获得解答
4. **代码质量**：AI提供的代码通常有较好的规范性

#### 注意事项
1. **不能完全依赖**：AI生成的代码需要仔细审查和测试
2. **理解原理很重要**：知其然更要知其所以然
3. **要有判断力**：AI的建议不一定完全正确，需要甄别
4. **保持独立思考**：AI是工具，最终决策还是要靠人

### 总结
AI是强大的辅助工具，但**人的思考和判断才是核心**。合理使用AI可以大幅提升开发效率，但不能替代对技术的深入理解和对产品的独立思考。
